Installation procedure overview

Here is an overview of the installation procedure. For more details, read the manual.

  1. Uncompress the archive distribution file (tangara-annotations-1.0.zip for instance).
  2. Create the java files with @LocalizableClass and @LocalizableMethod annotations
  3. Create the localize_<lang>.properties files containing the translation
  4. Create a localize.properties containing the list of translation languages.
  5. Run the javac command with the tangara-annotations.jar in the class-path and the option -A/path/to/localize.properties.

Requirements

  • Java SE 6 / JDK 1.6
  • ANT 1.7.1
