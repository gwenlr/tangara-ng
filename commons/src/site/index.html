Tangara commons library

General purpose

Tangara commons library is a Java utilities library used by Tangara project. It contains the following functionalities :

org.colombbus.tangara.commons.inject
A set of classes to load resources from properties files. Its provides also methods to inject resources into standard Swing components.
org.colombbus.tangara.commons.mvc
A set of class to help to create a Swing MVC application with action-methods and automated resource injection.

MVC pattern choice

Model-View-Controller or MVC design pattern is a well-known pattern in graphical applications. A lot of interpretations and refinement exist (MVP, MVC2, M+VC, etc.) about the responsabilities of each part (M,V,C), specialy on View and Controller parts.
In the tangara-commons, the responsabilities are the following:

Model
This part is in charge of the state of the data and its update. It is not aware of View nor Controller. It provides two types of access: direct access by methods to get or update its content, and event to follow automatically its content changes.
View
This part is in charge of display information to the user. It communicates with the Model through direct access for initialization purpose (read-only), and event for updating the display dynamically. It nevers updates directly the content of the model.
